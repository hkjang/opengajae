# 터미널 UI (TUI)

TUI는 터미널에서 Gateway에 연결하여 에이전트와 대화하고 상태를 모니터링하는 인터페이스입니다.

## 빠른 시작

```bash
# Gateway 시작 (별도 터미널)
openclaw gateway

# TUI 연결
openclaw tui
```

## 원격 Gateway 연결

```bash
# URL 직접 지정
openclaw tui --url ws://192.168.1.100:18789

# 토큰 인증
openclaw tui --url ws://remote:18789 --token YOUR_TOKEN

# 비밀번호 인증
openclaw tui --url ws://remote:18789 --password YOUR_PASSWORD
```

## 화면 구성

```
┌─────────────────────────────────────┐
│ 헤더: Gateway 상태, 에이전트, 모델  │
├─────────────────────────────────────┤
│                                     │
│ 채팅 로그                           │
│ - 사용자 메시지                     │
│ - 에이전트 응답                     │
│ - 도구 실행 카드                    │
│                                     │
├─────────────────────────────────────┤
│ 상태 라인: 세션, 토큰, 모드         │
├─────────────────────────────────────┤
│ 입력                                │
└─────────────────────────────────────┘
```

| 영역       | 내용                                    |
| ---------- | --------------------------------------- |
| 헤더       | Gateway 연결 상태, 현재 에이전트, 모델  |
| 채팅 로그  | 대화 히스토리, 스트리밍 응답, 도구 카드 |
| 상태 라인  | 세션 키, 토큰 사용량, 모드 표시         |
| 풋터       | 사용 가능한 키 바인딩                   |
| 입력       | 메시지 입력 영역                        |

## 키보드 단축키

| 키         | 동작                     |
| ---------- | ------------------------ |
| `Enter`    | 메시지 전송              |
| `Esc`      | 입력 취소/메뉴 닫기      |
| `Ctrl+C`   | 종료                     |
| `Ctrl+D`   | 종료                     |
| `Ctrl+L`   | 화면 지우기              |
| `Ctrl+G`   | 에이전트 피커            |
| `Ctrl+P`   | 세션 피커                |
| `Ctrl+O`   | 모델 피커                |
| `Ctrl+T`   | 사고 레벨 토글           |

## 슬래시 명령어

```
/help              # 도움말
/status            # Gateway 상태
/agent             # 에이전트 전환
/session           # 세션 전환
/model             # 모델 전환
/think [level]     # 사고 레벨 설정
/verbose [on|off]  # 상세 모드
/reasoning [mode]  # 추론 표시 모드
/usage [mode]      # 사용량 표시
/elevated [on|off] # 상승 권한
/deliver [on|off]  # 배달 모드
/new               # 새 세션
/abort             # 현재 실행 중단
/settings          # 설정
/exit              # TUI 종료
```

## 셸 명령어

`!` 접두사로 로컬 셸 명령어를 실행할 수 있습니다:

```
!ls -la
!git status
!npm test
```

## 배달 모드

다른 채널로 응답을 전달합니다:

```
/deliver telegram:123456789
```

## 옵션

| 옵션               | 설명                      | 기본값                  |
| ------------------ | ------------------------- | ----------------------- |
| `--url`            | Gateway WebSocket URL     | `ws://127.0.0.1:18789` |
| `--token`          | 인증 토큰                 | 자동 감지               |
| `--password`       | 인증 비밀번호             | -                       |
| `--session`        | 초기 세션 키              | main                    |
| `--deliver`        | 배달 대상                 | -                       |
| `--thinking`       | 사고 레벨                 | 기본값                  |
| `--timeout-ms`     | 요청 타임아웃 (ms)        | 300000                  |
| `--history-limit`  | 히스토리 표시 제한        | 50                      |

## 문제 해결

| 증상                    | 해결                                     |
| ----------------------- | ---------------------------------------- |
| 연결 안 됨              | Gateway 실행 확인, URL/포트 확인         |
| 출력 없음               | `--url` 확인, `--token` 확인             |
| 에이전트 피커 빈 상태   | 에이전트 설정 확인                       |
