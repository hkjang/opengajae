# Control UI

Control UI는 OpenClaw의 웹 기반 관리 대시보드입니다.

## 접근 방법

```bash
# 브라우저에서 열기
openclaw dashboard

# 직접 접속
# http://127.0.0.1:18789/
```

## 메인 탭

### Chat (채팅)

브라우저에서 에이전트와 직접 대화:

- **기능**
  - 마크다운 렌더링
  - 코드 하이라이팅
  - 이미지/파일 첨부
  - 드래그 앤 드롭
  - 복사/붙여넣기

- **단축키**
  - `Enter`: 메시지 전송
  - `Shift+Enter`: 줄바꿈
  - `Ctrl+L`: 세션 초기화
  - `Ctrl+K`: 명령어 팔레트

### Sessions (세션)

모든 활성 세션 관리:

- **기능**
  - 세션 목록 조회
  - 세션별 히스토리
  - 세션 초기화/삭제
  - 세션 내보내기

- **필터**
  - 채널별
  - 에이전트별
  - 최근 활동순

### Channels (채널)

채널 연결 상태 모니터링:

- **표시 정보**
  - 연결 상태
  - 마지막 활동
  - 오류 로그

- **액션**
  - 채널 활성화/비활성화
  - 재연결
  - QR 코드 표시 (WhatsApp)

### Config (설정)

설정 파일 편집:

- **기능**
  - JSON5 에디터
  - 구문 검사
  - 실시간 적용
  - 설정 백업/복원

- **섹션**
  - Gateway
  - Agents
  - Channels
  - Automation

### Logs (로그)

실시간 로그 뷰어:

- **기능**
  - 실시간 스트리밍
  - 레벨별 필터링
  - 검색
  - 내보내기

- **레벨 색상**
  - 🔵 Debug
  - 🟢 Info
  - 🟡 Warn
  - 🔴 Error

### Nodes (노드)

모바일 노드 관리:

- **기능**
  - 페어링 QR 코드
  - 노드 목록
  - 연결 상태
  - 페어링 해제

## 우측 사이드바

### 세션 정보

현재 채팅 세션의 정보:

- 세션 키
- 메시지 수
- 토큰 사용량
- 마지막 활동

### 도구 상태

에이전트가 사용 중인 도구:

- 활성 도구 목록
- 실행 중인 프로세스
- 브라우저 상태

## 테마

### 다크/라이트 모드

```
설정 → 테마 → 다크/라이트/시스템
```

### 폰트 크기

```
설정 → 표시 → 폰트 크기
```

## 모바일 지원

Control UI는 반응형 디자인:

- 태블릿/폰에서 사용 가능
- 터치 최적화
- 모바일 레이아웃

## 키보드 단축키

| 단축키         | 동작           |
| -------------- | -------------- |
| `Ctrl+K`       | 명령어 팔레트  |
| `Ctrl+L`       | 세션 초기화    |
| `Ctrl+,`       | 설정 열기      |
| `Ctrl+Shift+L` | 로그 패널      |
| `Escape`       | 현재 패널 닫기 |

## 문제 해결

### UI가 로드되지 않음

1. Gateway 상태 확인:

```bash
openclaw gateway status
```

2. 포트 확인:

```bash
curl http://127.0.0.1:18789/health
```

3. 브라우저 캐시 삭제

### 인증 오류

- 비밀번호 확인
- 쿠키 삭제 후 재로그인
- Tailscale 인증 확인

### 채팅이 응답하지 않음

1. 로그 탭에서 오류 확인
2. API 키 설정 확인
3. 모델 설정 확인
